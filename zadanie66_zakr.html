Pseudoelement :first-line pozwala na stylizację pierwszej
linii tekstu w elemencie. Jest to przydatne,
gdy chcemy nadać inną stylizację, taką jak inny kolor
lub rozmiar czcionki, pierwszej linii tekstu w bloku.
przykład:
p::first-line {
color: blue;
font-size: 120%;
}

Pseudoelementy :before i :after pozwalają na wstawienie
fikcyjnych elementów przed lub po zawartości rzeczywistego
elementu w HTML. Możemy użyć ich do dodania dekoracji,
ikon lub tekstu do elementów, bez konieczności zmiany samego HTML.
przykład:
p::before {
content: "Początek: ";
font-weight: bold;
} p::after {
content: " - Koniec.";
font-style: italic;
}

Pseudoelement :first-letter pozwala na stylizację
pierwszej litery tekstu w elemencie. Jest to przydatne,
gdy chcemy nadać pierwszej literze inny kolor, wielkość
czcionki lub styl.
przykład:
p::first-letter {
font-size: 150%;
color: red;
}
